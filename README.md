# Text Encryptor and Decryptor

A simple and efficient text encryption and decryption tool implemented in Java. This tool uses a symmetric encryption algorithm to securely encrypt and decrypt text.

## Features

- Encrypt any plain text into a secure format.
- Decrypt the encrypted text back to its original form.
- Easy-to-use command-line interface.
- Supports custom encryption keys.

## Requirements

- Java Development Kit (JDK) 8 or higher.
